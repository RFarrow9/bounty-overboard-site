# Bounty Overboard — site

Landing page for [Bounty Overboard](https://bounty-overboard.com), a top-down
pirate action game built in Godot 4.6.

Static site, no build step — GitHub Pages serves the repo root (`main` branch).

- `index.html` — the landing page
- `teaser.html` — the interactive "Take the helm" sail-around teaser
- `img/` — screenshots · `fonts/` — Pirata One (OFL) · `media/` — gameplay capture + poster
- `CNAME` — custom domain (`bounty-overboard.com`)

Deployed at **bounty-overboard.com** (domain registered at GoDaddy, DNS -> GitHub Pages).
